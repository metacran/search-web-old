About the CRAN package search

What?

This is a service that searches the descriptions of all active CRAN packages.

Why?

Searching is just easier than organizing.

How would you find all packages that are related to networks?. You can try searching manually on the CRAN web pages, or use CRAN task views, and these sometimes work, but often they don't. You cannot easily search in the package descriptions, and task views are not updated as frequently as the packages themselves. Taks views are also opinionated, and not all packages are (or even can be!) classified into the existing categories.

The existing search services do a good job in many cases, when you need to find a specific function, or need to know how to perform a simple task in . They are not the best at finding packages related to a given topic.

How?

We check CRAN frequently for changes, and update our database if a new package was admitted to CRAN, a package was updated, or archived. The database is automatically indexed by an Elastissearch server.

Command line version

You can also use this service from within R, using the seer package.
